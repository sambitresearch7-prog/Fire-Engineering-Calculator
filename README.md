# Fire-Engineering-Calculator
A free, interactive web tool for fire safety engineering. Features 8 essential modules including Window Radiation, Sprinkler &amp; Detector Activation (Alpert), Fire Severity (Law, CIB &amp; Eurocode), Occupant Movement, Compartment Burnout, FDS Mesh Size, and Smoke Layer Descent. Based on SFPE, ISO 16733-1, EN 1991-1-2, Buchanan &amp; Drysdale standards.
